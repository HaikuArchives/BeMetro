BeMetro 1.0


by Shigeru Kobayashi
http://www.ne.jp/asahi/geek/piano/

How to install ?

Unzip BeMetro.zip package. It's all :-)

How to use ?

Double click BeMetro icon, then wait several seconds (loading synthesizer sound file)..., it's ready. Usage is quite simple. Click 'Start' to start BeMetro, move slider to change tempo and click 'Stop' to stop.
